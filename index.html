ReactiveLock

Desenvolvido por Michel Oliveira

Disponível como biblioteca do ecossistema .NET no NuGet.org

O ReactiveLock é um sistema de controle de concorrência reativo e distribuído, com suporte ao Redis e otimizado para aplicações .NET modernas. Ele permite coordenar execuções entre múltiplas instâncias de forma leve, performática e resiliente.

➜ Ver no GitHub

Uma iniciativa com propósito

Este projeto nasceu da necessidade de controlar concorrência distribuída de forma segura e performática. É um convite à experimentação, contribuição e evolução conjunta. Se você trabalha com microsserviços, Redis ou sistemas concorrentes, vale a pena explorar.
